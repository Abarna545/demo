# Simple Responsive HTML Website

A clean, single-file responsive website built with pure HTML, CSS, and a small amount of JavaScript. Perfect for landing pages, portfolios, or quick demos.

## Features
- Fully responsive layout  
- Modern UI with no external libraries  
- Mobile navigation menu  
- Demo contact form with basic validation  
- Lightweight and fast (all-in-one HTML file)

## How to Use
1. Download or clone the repository.  
2. Open `index.html` in any browser to view the site.  
3. Edit the text, styles, or sections directly inside `index.html`.  
4. Deploy by uploading the file to GitHub Pages, Netlify, Vercel, or any static hosting service.
